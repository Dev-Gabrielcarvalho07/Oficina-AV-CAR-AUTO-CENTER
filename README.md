# AV CAR AUTO CENTER — API Back-end

## Pré-requisitos
- Java 17+
- Maven 3.8+

## Configuração do banco
Copie o arquivo de exemplo e preencha com suas credenciais:
```
cp src/main/resources/database.properties.example src/main/resources/database.properties
```

## Compilar
```
mvn compile
```

## Executar
```
mvn exec:java
```
